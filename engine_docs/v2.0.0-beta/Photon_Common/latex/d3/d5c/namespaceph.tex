The returned memory resource will follow the life time of {\ttfamily std\+::unique\+\_\+ptr}. Note that the memory allocated by this function is raw memory--placement new is required before any use of the memory content, otherwise it is UB by C++ standard (for non-\/implicit-\/lifetime types). For implicit-\/lifetime types, accessing the raw memory without placement new has defined behavior after C++20.

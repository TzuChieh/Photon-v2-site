◆ forEachUsableObject()

template<typename PerObjectOperation >
void ph::editor::DesignerScene::forEachUsableObject ( PerObjectOperation  op) const
inline

By usable, it means the object can be called/used from main thread. For an object to be "usable", it must not be in any erroneous state, initialized on main thread (isInitialized(obj) == true), and not being an orphan.

Parameters
opInstance that can be invoked as bool (DesignerObject* obj), where obj is an object satisfying the "usable" requirement. The return value indicates whether the iteration process should continue.
Note
Can only be called from main thread.

Detailed Description

template<typename Item, CHandleDispatcher Dispatcher = THandleDispatcher<TWeakHandle<Item>>, typename ItemInterface = Item>
class ph::editor::TItemPool< Item, Dispatcher, ItemInterface >

A general item pool.

Template Parameters
ItemType of the stored datum. Can be any class or primitive type. Must be move constructible.
DispatcherThe dispatcher to use for this pool. Item pool requires the handle type to be "compatible" to Item, see the definition of TCompatibleHandleType for more detail.

template<typename Item , CHandleDispatcher Dispatcher = THandleDispatcher<TWeakHandle<Item>>, typename ItemInterface = Item>

using ph::editor::TItemPool< Item, Dispatcher, ItemInterface >::TCompatibleHandleType = TWeakHandle<ItemType, Index, Generation>

If Item is a polymorphic type, then Item itself along with all its bases can form a valid handle type. These compatible handles can be used for accessing items stored in the pool (items obtained have the same item type indicated by the handle, for type safety).

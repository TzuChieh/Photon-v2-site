Detailed Description

Graphics memory allocation helper. The arena is allocating lazily, i.e., constructing and copying an arena instance won't allocate any memory, the actual allocation occurs when memory is explicitly requested. In addition to memory reusability, this class also provides various fallback mechanisms to ensure every memory request can be fulfilled.

\subsubsection*{template$<$std\+::size\+\_\+t NUM\+\_\+\+BUFFERS, typename \mbox{\hyperlink{namespaceph_1_1editor_a8416c0e2549730bba9233b7ab6413c68ae1e1d3d40573127e9ee0480caf1283d6}{R}}, typename... Args$>$\newline
class ph\+::editor\+::\+TFrame\+Worker\+Thread$<$ NUM\+\_\+\+BUFFERS, R(\+Args...)$>$}
A worker thread that helps to develop the concept of frame-\/to-\/frame work which is executed on another thread. For unbuffered frames, you can specify {\ttfamily NUM\+\_\+\+BUFFERS = 1}, though one might consider to use {\ttfamily \doxylink{classph_1_1editor_1_1_t_unbuffered_frame_worker_thread}{TUnbuffered\+Frame\+Worker\+Thread}} for this as it is specialized for this case and can offer better performance. Note that in {\ttfamily \doxylink{classph_1_1editor_1_1_t_frame_worker_thread}{TFrame\+Worker\+Thread}}, works will not be processed until {\ttfamily \doxylink{classph_1_1editor_1_1_t_frame_worker_thread_3_01_n_u_m___b_u_f_f_e_r_s_00_01_r_07_args_8_8_8_08_4_a2c961f276e1af646dc807d50697f66af}{end\+Frame()}} is called, while in {\ttfamily \doxylink{classph_1_1editor_1_1_t_unbuffered_frame_worker_thread}{TUnbuffered\+Frame\+Worker\+Thread}} works will start being processed right after {\ttfamily \doxylink{classph_1_1editor_1_1_t_frame_worker_thread_3_01_n_u_m___b_u_f_f_e_r_s_00_01_r_07_args_8_8_8_08_4_ae01fc47788fe7c65a8300d2d66fd9720}{begin\+Frame()}}.

Regarding thread safety notes\+:

Parent Thread\+: Thread that starts the worker thread (by calling {\ttfamily \doxylink{classph_1_1editor_1_1_t_frame_worker_thread_3_01_n_u_m___b_u_f_f_e_r_s_00_01_r_07_args_8_8_8_08_4_aeccac1b02007ffec04f9a1ac7a8ea564}{start\+Worker()}}). Worker Thread\+: Thread that processes/consumes work, only one worker thread will be spawned. Producer Thread\+: Thread that adds/produces work, can be one or more. For a non-\/parent producer thread, it must synchronize with {\ttfamily \doxylink{classph_1_1editor_1_1_t_frame_worker_thread_3_01_n_u_m___b_u_f_f_e_r_s_00_01_r_07_args_8_8_8_08_4_ae01fc47788fe7c65a8300d2d66fd9720}{begin\+Frame()}} and {\ttfamily \doxylink{classph_1_1editor_1_1_t_frame_worker_thread_3_01_n_u_m___b_u_f_f_e_r_s_00_01_r_07_args_8_8_8_08_4_a2c961f276e1af646dc807d50697f66af}{end\+Frame()}}. Thread Safe\+: Can be used on any thread. Anything that has no thread safety notes\+: It is {\bfseries{NOT}} thread safe.

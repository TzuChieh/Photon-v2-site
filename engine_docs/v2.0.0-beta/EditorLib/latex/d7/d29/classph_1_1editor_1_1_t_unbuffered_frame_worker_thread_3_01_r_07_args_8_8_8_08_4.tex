\subsubsection*{template$<$typename \mbox{\hyperlink{namespaceph_1_1editor_a8416c0e2549730bba9233b7ab6413c68ae1e1d3d40573127e9ee0480caf1283d6}{R}}, typename... Args$>$\newline
class ph\+::editor\+::\+TUnbuffered\+Frame\+Worker\+Thread$<$ R(\+Args...)$>$}
A worker thread that helps to develop the concept of frame-\/to-\/frame work which is executed on another thread. Unlike its variant, {\ttfamily \doxylink{classph_1_1editor_1_1_t_frame_worker_thread}{TFrame\+Worker\+Thread}}, this class has no buffer and works added will be processed as soon as possible (no need to wait for the call to \doxylink{classph_1_1editor_1_1_t_unbuffered_frame_worker_thread_3_01_r_07_args_8_8_8_08_4_a403d0bbbc93e2805fbdcdd159e847585}{end\+Frame()}).

Regarding thread safety notes\+:

Parent Thread\+: Thread that starts the worker thread (by calling {\ttfamily \doxylink{classph_1_1editor_1_1_t_unbuffered_frame_worker_thread_3_01_r_07_args_8_8_8_08_4_a825e523a69770b1e2eb06d6360bd5d10}{start\+Worker()}}). Worker Thread\+: Thread that processes/consumes work, only one worker thread will be spawned. Thread Safe\+: Can be used on any thread. Anything that has no thread safety notes\+: It is {\bfseries{NOT}} thread safe.

ctor call {\ttfamily \doxylink{classph_1_1editor_1_1_t_unbuffered_frame_worker_thread_3_01_r_07_args_8_8_8_08_4_a825e523a69770b1e2eb06d6360bd5d10}{start\+Worker()}} one or more calls to\+:

\item any number of calls to {\ttfamily \doxylink{classph_1_1editor_1_1_t_unbuffered_frame_worker_thread_3_01_r_07_args_8_8_8_08_4_a916b54879baaff1252b97aabc0316510}{add\+Work()}}
\item possibly call {\ttfamily \doxylink{classph_1_1editor_1_1_t_unbuffered_frame_worker_thread_3_01_r_07_args_8_8_8_08_4_a9b9732422e177dffab30f3482e26fac6}{request\+Worker\+Stop()}}

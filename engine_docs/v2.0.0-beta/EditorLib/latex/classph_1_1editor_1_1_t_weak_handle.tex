Get the next generation given a current generation. 

\begin{DoxyNote}{Note}
We do not provide a method to directly "{}tick"{} a handle\textquotesingle{}s generation since that would partially defeat the purpose of using a generation counter--that may encourage the user to tick a handle\textquotesingle{}s generation until it is valid. Getting the next generation, however, is useful for many library routines so we provide a static method here.

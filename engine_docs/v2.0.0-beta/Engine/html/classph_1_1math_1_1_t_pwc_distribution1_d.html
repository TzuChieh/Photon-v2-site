Detailed Description

template<typename T>
class ph::math::TPwcDistribution1D< T >

A 1-D piecewise constant distribution of floating-point type T. The sample weights can be seen as a histogram, and samples are drawn according to each column's relative heights. Each generated sample is guaranteed to have a non-zero PDF.

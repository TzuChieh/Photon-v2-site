inline

Calculates trigonometries for the \( \phi \) angle in spherical coordinates.

Remarks
The default coordinate system in Photon is right-handed, y-up. As a result, \( \phi = 0 \) starts on z-axis, and \( \phi = \pi / 2 \) is on x-axis.

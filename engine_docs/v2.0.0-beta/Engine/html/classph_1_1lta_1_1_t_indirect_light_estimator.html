Constructing path with BSDF sampling and estimate lighting with both BSDF sampling and next-event estimation. This method is effectively the same as TDirectLightEstimator::bsdfSamplePathWithNee() if input path length is in range [1, 1].

Parameters
minPathLengthThe minimum length of the path to construct.
maxPathLengthThe maximum length of the path to construct (inclusive).
out_LoSampled outgoing energy from X. The energy is for the specified path length range only.
rrBeginPathLengthWhen to start using russian roulette. If 0, russian roulette will be performed right away, before any sampling take place.
Returns
Whether output parameters are usable. If false is returned, the sample should still be treated as valid, albeit its contribution is effectively zero.

Detailed Description

template<math::EColorSpace COLOR_SPACE>
class ph::TColorPixelTexture2D< COLOR_SPACE >
Template Parameters
COLOR_SPACESource color space. The color space of the contained pixel data. If is math::EColorSpace::Unspecified, then raw color data is used (without any space transformations). Using math::EColorSpace::Unspecified when the engine is in spectral mode may fallback to math::EColorSpace::Linear_sRGB if a direct conversion is impossible.

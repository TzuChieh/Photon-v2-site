Detailed Description

template<ESidednessPolicy POLICY>
class ph::lta::TDirectLightEstimator< POLICY >

Estimate direct lighting for a surface point. This is a lightweight helper type for estimating direct lighting. Do not think "direct light" as lighting from a directional light source, it means the first-bounce lighting for any surface point, and the surface point can be the N-th one in a path.

Template Parameters
POLICYThe default sidedness policy to use if none can be inferred from the inputs.

Parameters

out_LeThe sampled emitted energy of another surface. Does not contain any weighting.
out_XReturns the surface that is sampled. It is not necessary an energy-emitting surface (as out_Le could be 0 if out_X is requested).
Returns
Whether output parameters are usable. If false is returned, the sample should still be treated as valid, albeit its contribution is effectively zero.

Detailed Description

A GI estimator using pure path tracing.

BVPT: Backward Vanilla Path Tracing.

This estimator is the basic version of path tracing. Rays shot from the receiver, bouncing around the scene until an emitter is hit, gather its emitted power and keep bouncing... Most online tutorials implemented this GI algorithm. Since this estimator is relatively simple but still unbiased, it is good for ground truth rendering if the correctness of another estimator is in doubt.

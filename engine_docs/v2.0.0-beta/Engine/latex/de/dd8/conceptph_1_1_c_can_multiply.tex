\doxysubsection{Detailed Description}
Check if instances of types can be multiplied together. 

Checks whether the instances of the involved types can be multiplied together and is capable of assigning the result into an instance of the third type, i.\+e., able to do C = A \texorpdfstring{$\ast$}{*} B with corresponding instances.

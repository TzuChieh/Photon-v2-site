\subsubsection*{template$<$typename R, typename... Args, std\+::size\+\_\+t MIN\+\_\+\+SIZE\+\_\+\+HINT$>$\newline
class ph\+::function\+\_\+detail\+::\+TFunction$<$ R(\+Args...), MIN\+\_\+\+SIZE\+\_\+\+HINT $>$}
Lightweight callable target wrapper. 

This type is a thin wrapper around stateless callable targets such as free function, method, functor and lambda. For methods, the instance must outlive the associated {\ttfamily \doxylink{classph_1_1function__detail_1_1_t_function}{TFunction}}. For functors and lambdas, they must be stateless (no member variable/no capture) or small enough in size. Unlike the standard {\ttfamily std\+::function} which offers little guarantees, this type is guaranteed to be cheap to construct, copy, destruct and small in size (minimum\+: 2 pointers for {\ttfamily MIN\+\_\+\+SIZE\+\_\+\+HINT} == 16, may vary depending on the platform). Calling functions indirectly through this type adds almost no overhead.


\begin{DoxyTemplParams}{Template Parameters}
{\em MIN\+\_\+\+SIZE\+\_\+\+HINT} & Hint to the minimum size of \doxylink{classph_1_1function__detail_1_1_t_function}{TFunction}. Larger size may correspond to bigger internal buffer. A size of 0 will result in the most compact \doxylink{classph_1_1function__detail_1_1_t_function}{TFunction} possible. This is the lower-\/bound of the size of \doxylink{classph_1_1function__detail_1_1_t_function}{TFunction}, and the actual size cannot be guaranteed. Normally, most supported callable target types will be guaranteed to be storable regardless of this parameter, except for non-\/empty functors.\\

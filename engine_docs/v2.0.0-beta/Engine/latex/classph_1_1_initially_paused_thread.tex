Creates a thread with a function to execute. The function will not execute immediately. The thread is in paused state initially and the execution can be started by calling \doxylink{classph_1_1_initially_paused_thread_aacc8da38d38fa892417ae50197b0995e}{start()}. 


\begin{DoxyParams}{Parameters}
{\em func} & Functor to be called by the thread. It is copied/moved by value and stored in the thread. \\
\hline
{\em args} & Arguments passed to the functor. They are copied/moved by value and stored in the thread. \\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
{\ttfamily func} and {\ttfamily args} are passed the same way as {\ttfamily std\+::thread}, which means to pass a reference argument to the thread, it has to be wrapped (e.\+g., with {\ttfamily std\+::ref} or {\ttfamily std\+::cref}). 
\end{DoxyNote}

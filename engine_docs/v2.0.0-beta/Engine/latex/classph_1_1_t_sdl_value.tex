\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T, typename Owner$>$\newline
class ph\+::\+TSdl\+Value$<$ T, Owner $>$}
Abstractions for a SDL value type. 

A default value can be specified for instances of this type. If the default value is not explicitly provided (by {\ttfamily \doxylink{classph_1_1_t_sdl_value_a8fd4f58c79486f537e2b5be3e5fea90d}{default\+To()}} or other means), default value will be the {\ttfamily value initialization} of the type {\ttfamily T} (basically a mix of default and zero initialization). If default value is not wanted, use {\ttfamily \doxylink{classph_1_1_t_sdl_value_aaf3a5d8f0706185fcdc89dce00e57dd7}{no\+Default()}}.

{\ttfamily value initialization}\+: \href{https://en.cppreference.com/w/cpp/language/value_initialization}{\texttt{ https\+://en.\+cppreference.\+com/w/cpp/language/value\+\_\+initialization}}

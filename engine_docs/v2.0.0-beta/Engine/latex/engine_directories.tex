\chapter{Engine Directories}
\hypertarget{engine_directories}{}\label{engine_directories}\index{Engine Directories@{Engine Directories}}
An overview of engine directory structure.

Engine\textquotesingle{}s source code is organized into directories based on the functionalities they provide. The information listed here might help you to find what you are looking for if a quick search gives no useful results.\hypertarget{engine_directories_root}{}\doxysection{\texorpdfstring{Root Directories}{Root Directories}}\label{engine_directories_root}
Home of header and source files that provides functionalities Photon has to offer.

Generally speaking, the \doxylink{dir_856524284ebe840938865dc061f982fb}{Include} folder contains public APIs for the engine, while the \doxylink{dir_74389ed8173ad57b461b9d623a1f3867}{Source} folder contains its implementation. For those who want direct access to all engine functionalities, you can choose to compile and link code from both \doxylink{dir_856524284ebe840938865dc061f982fb}{Include} and \doxylink{dir_74389ed8173ad57b461b9d623a1f3867}{Source}.\hypertarget{engine_directories_data_containers}{}\doxysection{\texorpdfstring{Standalone/\+Composite Data Containers}{Standalone/\+Composite Data Containers}}\label{engine_directories_data_containers}
These directories contains raw data descriptions for a scene. They will contribute to the final image after being cooked.

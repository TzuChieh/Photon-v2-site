\doxysubsection{Detailed Description}
A group of cooked data that represent the visible part of the scene at a specific time. This data block do not persist throughout the rendering process. After cooking is done, all cooked data should be properly interlinked and {\ttfamily \doxylink{classph_1_1_transient_visual_element}{Transient\+Visual\+Element}} will be cleaned up.

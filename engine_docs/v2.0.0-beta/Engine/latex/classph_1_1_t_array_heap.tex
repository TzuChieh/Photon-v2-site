A fixed size heap backed by an array. The container inherits the properties of a fixed size array of type {\ttfamily T}. The container keeps all {\ttfamily N} elements alive until its lifetime ends, even if elements are popped/removed from the stack. It is guaranteed that no dynamic memory allocation takes place.

\doxysubsection{Detailed Description}
Dielectric-\/dielectric interface Fresnel effect. 

The formulae used are dielectric-\/specific approximations made by Brent Burley \cite{Burley:2015:Extending}, which provide some improvement over the approximation originally done by Schlick \cite{Schlick:1994:BRDF}. See the implementation of {\ttfamily \doxylink{classph_1_1_schlick_approx_dielectric_fresnel_a264fc2b030cf012b0439092e75705249}{calc\+Reflectance()}} for more details.

\doxysubsection{Detailed Description}
Raw data storage of a picture. Has limited ability on pixel data manipulation. It is recommended to convert to {\ttfamily \doxylink{classph_1_1_t_frame}{TFrame}} via {\ttfamily \doxylink{classph_1_1_picture_data_af7ff646132e6c0df5a6ea02f486b9994}{to\+Frame()}} for operations on pixel data.

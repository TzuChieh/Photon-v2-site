\doxysubsection{Detailed Description}
Contains main C APIs of the render engine. 

To correctly use Photon-\/v2 API, please read the following notes\+:


\begin{DoxyItemize}
\item \doxylink{ph__c__core_8h_a02c4de0a9c0475f0ac8dd39b53b1a238}{ph\+Init()} and \doxylink{ph__c__core_8h_a260bd41a3522416e9f93451ce933071d}{ph\+Exit()} must be called before and after the use of the API, and should be called on the same thread.
\item ph\+Create/\+Delete$<$\+X$>$() and ph\+Async$<$\+X$>$() functions can be used in a multithreaded environment, namely, they are thread-\/safe. An exception would be that create and delete should be called from the same thread.
\item Resources created by ph\+Create$<$\+X$>$() cannot be manipulated concurrently. Any function requiring some resource ID inputs (except ph\+Async$<$\+X$>$() functions) is considered a resource manipulating operation on those resources. 
\end{DoxyItemize}

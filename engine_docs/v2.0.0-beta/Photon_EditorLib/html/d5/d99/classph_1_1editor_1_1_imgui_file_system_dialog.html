Path ph::editor::ImguiFileSystemDialog::getSelectedTarget ( ) const

Get the selected filesystem target. The result can be either the selected directory or a full path to the selected item. This method effectively combines the result of getSelectedDirectory() and getSelectedItem(). Call after dialogClosed() for a complete result. If the dialog is not closed, intermediate result will be returned.

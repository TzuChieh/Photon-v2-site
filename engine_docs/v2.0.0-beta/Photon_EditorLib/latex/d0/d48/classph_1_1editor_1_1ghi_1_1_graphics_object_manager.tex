Manages the creation and deletion of graphics-\/related resource objects. All {\ttfamily create\texorpdfstring{$<$}{<}XXX\texorpdfstring{$>$}{>}()} and {\ttfamily delete\texorpdfstring{$<$}{<}XXX\texorpdfstring{$>$}{>}()} along with some data manipulation methods are thread safe as long as they are called within the lifetime of current graphics context.

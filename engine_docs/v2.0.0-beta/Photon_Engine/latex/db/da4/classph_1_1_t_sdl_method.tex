\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Method\+Struct, typename Target\+Type$>$\newline
class ph\+::\+TSdl\+Method$<$ Method\+Struct, Target\+Type $>$}
SDL binding type for a canonical SDL method. 

For a method to be callable in SDL, it must be defined as a C++ functor that takes an instance of {\ttfamily Target\+Type} as its only input. Other arguments to the method should be accessed via the functor\textquotesingle{}s member variables. Each member variables then can be binded much like the members of a canonical SDL resource.

\doxysubsection{Detailed Description}
Maps roughness value in \mbox{[}0, 1\mbox{]} to the alpha parameter in NDF. 

Maps roughness value in \mbox{[}0, 1\mbox{]} to alpha parameter used by many , microfacet normal distribution functions. Generally, the mapping can be quite beneficial for artists since directly consider roughness as alpha value often does not result in linearly varying specular highlight across equally spaced roughness values. Strangely, the original paper\+: \doxylink{classph_1_1_microfacet}{Microfacet} Models for Refraction through Rough Surfaces by Walter et al. does not include such mapping for GGX distribution, only the ones for other kinds of distribution.

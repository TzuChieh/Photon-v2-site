\doxysubsection{Detailed Description}
Describes enum in SDL. 

Note that it is highly encouraged to follow several conventions while using SDL enum\+:
\begin{DoxyItemize}
\item Define an entry with value = 0 to indicate a default/unspecified state.
\item Define only unique enum entries--no identical enum values or enum names. 
\end{DoxyItemize}

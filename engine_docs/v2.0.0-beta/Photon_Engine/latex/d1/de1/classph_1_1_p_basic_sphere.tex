{\footnotesize\ttfamily bool ph\+::\+PBasic\+Sphere\+::may\+Overlap\+Volume (\begin{DoxyParamCaption}\item[{const \mbox{\hyperlink{classph_1_1math_1_1_t_a_a_b_b3_d}{math\+::\+AABB3D}} \&}]{volume }\end{DoxyParamCaption}) const\hspace{0.3cm}{\ttfamily [override]}, {\ttfamily [virtual]}}



Conservatively checks whether this object overlaps a volume. 

By conservative, it means {\bfseries{true can be returned even though the object does not overlap the volume}}; but if it actually does, {\bfseries{true must be returned}}. The default implementation performs conservative intersecting test using the AABB calculated by \doxylink{classph_1_1_p_basic_sphere_a75e188eb9cbc9915c9072e741495ec30}{calc\+AABB()}. Although false-\/positives are allowed for this method, providing an implementation with higher accuracy is benefitial for many algorithms used by the renderer. The test generally considers the underlying shape as {\bfseries{hollow}} (for closed shape), while the volume is {\bfseries{solid}}.

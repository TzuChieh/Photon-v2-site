Copy a file or directory recursively to another destination. If {\ttfamily src\+Path} is a file, {\ttfamily dst\+Path} will be interpreted as a file and the method is equivalent to calling {\ttfamily \doxylink{classph_1_1_filesystem_a34b2918123bbb51bcecbcb2a7f0ecd72}{copy\+File()}}. If {\ttfamily src\+Path} is a directory, {\ttfamily dst\+Path} will be interpreted as a directory. For directory copying, contents in {\ttfamily src\+Path} will be copied to {\ttfamily dst\+Path}. Missing directories are automatically created.

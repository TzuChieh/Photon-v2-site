Conservatively checks whether this object overlaps a volume. 

By conservative, it means {\bfseries{true can be returned even though the object does not overlap the volume}}; but if it actually does, {\bfseries{true must be returned}}. The default implementation performs conservative intersecting test using the AABB calculated by \doxylink{classph_1_1_t_meta_injection_primitive_a9d726a028a60e5cc0f2fd4434926ae7f}{calc\+AABB()}. Although false-\/positives are allowed for this method, providing an implementation with higher accuracy is benefitial for many algorithms used by the renderer. The test generally considers the underlying shape as {\bfseries{hollow}} (for closed shape), while the volume is {\bfseries{solid}}.

\doxysubsection{Detailed Description}
SDL struct do not provide interface for initializing instances to default values such as the one provided by SDL class ({\ttfamily \doxylink{classph_1_1_sdl_class_adac95fa0dea658b8abdc876122837318}{Sdl\+Class\+::init\+Default\+Resource()}}). The rationale behind this is that SDL struct is designed to be a simple grouped data carrier without the burden of inheritance and member functions. For efficiency and ease of reuse, initializing instances to default values requires a concrete type (the exact type of the struct). Implementations that wish to support this feature should use define a method with signature {\ttfamily init\+Default\+Struct(T\& instance) const} where T is the concrete type of the struct.

{\ttfamily \doxylink{classph_1_1_t_sdl}{TSdl}} provides a simplified interface for initializing instances to default values.

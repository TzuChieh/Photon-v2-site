Conservatively checks whether this object overlaps a volume. 

By conservative, it means {\bfseries{true can be returned even though the object does not overlap the volume}}; but if it actually does, {\bfseries{true must be returned}}. The default implementation performs conservative intersecting test using the AABB calculated by \doxylink{classph_1_1_transformed_primitive_a80d4ff93de46b1a99e5c75de26be49bc}{calc\+AABB()}. Although false-\/positives are allowed for this method, providing an implementation with higher accuracy is benefitial for many algorithms used by the renderer. The test generally considers the underlying shape as {\bfseries{hollow}} (for closed shape), while the volume is {\bfseries{solid}}.

\doxysubsection{Detailed Description}
Test whether {\ttfamily T} is an implicit-\/lifetime type. This concept is not exactly {\ttfamily std\+::is\+\_\+implicit\+\_\+lifetime\+\_\+v}, it is slightly permissive in the sense that true implicit-\/lifetime type cannot have a user-\/provided destructor, and we are only testing if it is trivially destructible. See P2674\+R0 "{}\+A trait for implicit lifetime types"{} for more details.

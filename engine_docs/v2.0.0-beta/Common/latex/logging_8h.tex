\doxysubsection{Detailed Description}
Logging functions. 

Note on loggers\+: All logging functionalities are thread-\/safe when using pre-\/defined macros. It is not advisible to log in class dtor, especially for static instances as they may live outside of engine\textquotesingle{}s lifetime. Also make sure not to call any logging functions when the logger is not initialized.
